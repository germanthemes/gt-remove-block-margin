Remove top and bottom spacing for any Gutenberg block.

== Description ==

Some Gutenberg Blocks or WordPress Themes add unwanted space around blocks. This small plugin allows you to remove the top and / or bottom spacing for any Gutenberg block. This is especially helpful to remove top or bottom spacing of Heading and Group blocks.

The settings are added to the Advanced tab in the Block Sidebar. See screenshot below.

== Installation ==

1. Activate the plugin
2. Create a new post or page and edit a block.
3. In the Advanced Tab of the Block Sidebar, you can find the settings to remove block margins.
3. Please also see the screenshot and our [Documentation](https://germanthemes.de/en/plugin/gt-remove-margin-documentation/)
